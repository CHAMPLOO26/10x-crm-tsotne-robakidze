This project is a simple Customer Relationship Management (CRM) system built with HTML, CSS and JavaScript.
The application allows users to register, log in, and manage customer information.
User data and CRM information are stored in the browser using LocalStorage.

# 10X CRM

A responsive CRM web application built with HTML, CSS, and vanilla JavaScript.

## Features

- User registration with form validation
- Login and persistent user session
- Protected pages for authenticated users
- Dashboard with live clock, statistics, pipeline overview, and recent clients
- Client search, status filtering, and sorting
- Add and delete clients
- Client details page
- Add and delete client notes
- Editable user profile
- Persistent dark mode
- Data persistence with Local Storage

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage
- DummyJSON API

## Local Storage Keys

- `crm_users` — registered users
- `crm_session` — current logged-in user session
- `crm_clients` — CRM client data
- `crm_theme` — selected light or dark theme

## Pages

- `index.html` — Login page
- `signup.html` — Registration page
- `dashboard.html` — Dashboard
- `clients.html` — Client list and management
- `client-details.html` — Individual client details and notes
- `profile.html` — User profile
