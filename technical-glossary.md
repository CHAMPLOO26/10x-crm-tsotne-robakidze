# Technical Glossary

## 1. Authentication

**English:** Authentication is the process of confirming a user's identity before allowing access to protected content.  
**ქართული:** ავტორიზაცია ნიშნავს იმის შემოწმებას, ვინ არის მომხმარებელი, სანამ მას დაცულ გვერდზე შევუშვებთ.

## 2. Session

**English:** A session stores information about the user who is currently logged in.  
**ქართული:** სესია ინახავს ინფორმაციას იმ მომხმარებელზე, რომელიც ამ მომენტში დალოგინებულია.

## 3. Validation

**English:** Validation checks whether user input follows the required rules before the data is saved.  
**ქართული:** ვალიდაცია ამოწმებს, სწორად არის თუ არა შევსებული ველები მონაცემის შენახვამდე.

## 4. Local Storage

**English:** Local Storage is browser storage that keeps data after a page refresh or browser restart.  
**ქართული:** Local Storage არის ბრაუზერის საცავი, სადაც მონაცემი refresh-ის შემდეგაც რჩება.

## 5. Fetch

**English:** Fetch is a browser function used to request data from an API.  
**ქართული:** Fetch არის ბრაუზერის ფუნქცია, რომლითაც API-დან მონაცემები მოგვაქვს.

## 6. API

**English:** An API is an interface that allows one application to request or send data to another system.  
**ქართული:** API არის საშუალება, რომ ერთმა პროგრამამ მეორე სისტემიდან მონაცემი მიიღოს ან გაუგზავნოს.

## 7. JSON

**English:** JSON is a text format commonly used to exchange structured data between applications.  
**ქართული:** JSON არის ტექსტური ფორმატი, რომლითაც ობიექტების და მასივების მსგავსი მონაცემი გადადის პროგრამებს შორის.

## 8. Event Listener

**English:** An event listener waits for an action, such as a click, input, or form submission.  
**ქართული:** Event listener ელოდება მომხმარებლის მოქმედებას, მაგალითად click-ს, ტექსტის ჩაწერას ან ფორმის გაგზავნას.

## 9. State

**English:** State is the current data and condition of an application at a specific moment.  
**ქართული:** State არის აპლიკაციის მიმდინარე მდგომარეობა, მაგალითად არჩეული ფილტრი ან კლიენტების სია.

## 10. DOM

**English:** The DOM is a browser representation of an HTML page that JavaScript can read and change.  
**ქართული:** DOM არის HTML გვერდის ბრაუზერში შექმნილი სტრუქტურა, რომლის წაკითხვა და შეცვლა JavaScript-ს შეუძლია.
